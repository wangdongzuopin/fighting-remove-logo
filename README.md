# fighting-remove-logo

小蜜蜂水印去除工具，现已适配为 Chrome 扩展程序。

## 功能

- 上传图片并框选水印区域后，按原格式导出并保持原分辨率
- 处理结果会直接替换预览图，可继续二次处理
- 支持一键恢复到第一版上传原图

## 本地加载

1. 打开 Chrome，进入 `chrome://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前项目根目录 `fighting-remove-logo`

## 打包文件

- 可分发压缩包：`dist/fighting-remove-logo-chrome-extension.zip`
- 扩展入口清单：`manifest.json`
